# 命名与名词表（novel-naming-glossary-managing）

## 目标

- 让读者不迷路：同一个东西只用一种主写法，别名有登记。
- 让作者不崩溃：长篇后期检索与回修靠名词表救命。

## 权威文件

- `novel/bible/glossary.md`
- 相关 bible 实体文件：`novel/bible/**`

## 最短路径工作流

1. 明确要处理的对象：人物/地点/势力/体系/道具/境界术语。
2. 选择主写法（读者最容易记住、最不易混淆）。
3. 维护别名：把旧称呼/外号/简称写进 `aliases` 或 glossary 的别名列。
4. 标注首次出现章节：方便回读与检查（例如 `ch-012`）。
5. 检查冲突：
   - 是否与已有名词同音同形导致误读
   - 是否与同类术语风格不一致（例如境界命名体系乱）
6. 如涉及新增实体：同步调用 `novel-bible-managing` 创建对应条目（带 ID）。

## 输出要求

- 更新后的 `novel/bible/glossary.md`（必要时更新相关实体文件）
